# Generador de balances de situación

Aplicación educativa en Python/Streamlit para generar balances de situación ficticios siguiendo la normativa contable española.

## Características

- Generación de balances por tamaño de empresa (micro, pyme, mediana, grande).
- Validación automática de la ecuación patrimonial.
- Cálculo de ratios básicos (fondo de maniobra, liquidez, solvencia, endeudamiento).
- Pensado para modo estudiante, examen y profesor (extensible).

## Requisitos

- Python 3.12
- Ver `requirements.txt`.

## Ejecución rápida en modo consola

```bash
pip install -r requirements.txt
python main.py
```
